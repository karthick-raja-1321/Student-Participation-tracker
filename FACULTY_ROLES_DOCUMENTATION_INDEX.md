## üéì Example: Adding Faculty with Both Roles

### In the UI:
1. Click "Add Faculty"
2. Fill: First Name: "Rajesh", Last Name: "Kumar", Email: "rajesh@college.edu"
3. Check "Class Advisor" ‚Üí Year: 2, Section: A
4. Check "Innovation Coordinator"
5. Click "Save"
6. **Result:** Faculty appears with "2 A, Innovation Coordinator" in roles column

### In Excel:
1. Download template
2. Fill: Rajesh | Kumar | rajesh@college.edu | ... | 2 CSE A | TRUE
3. Upload file
4. **Result:** Faculty created with both roles

### In Database:
```json
{
  "isClassAdvisor": true,
  "classAdvisorClasses": [{ "year": 2, "section": "A", "departmentId": "..." }],
  "isInnovationCoordinator": true
}
```
